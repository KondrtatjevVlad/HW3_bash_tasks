# Отчет по выполнению Bash-заданий

## Задание 1

**Цель:** написать Bash-скрипт, который:

* Создает список всех файлов в текущей директории и выводит их тип
* Проверяет наличие файла, переданного как аргумент
* Выводит список файлов с правами доступа

**Пример запуска скрипта:**


$ ./task1.sh test.txt
Список файлов и их тип:
task1.sh — файл
test.txt — файл
Файл 'test.txt' существует.

Файлы и их права доступа:
-rw-r--r-- 1 Vlad 197609 883 Nov  1 20:11 task1.sh
-rw-r--r-- 1 Vlad 197609 0 Nov  1 20:11 test.txt
