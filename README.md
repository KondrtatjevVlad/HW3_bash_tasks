## Задание 5

**Запуск скрипта:**

Запускаются 3 команды на фоне с разными временными интервалами ( к примеру sleep 5 & sleep 10 & sleep 15 &)

Выводится сообщение о запуске фоновых задачах. Команда jobs показывает список запущенных задач.

Командой fg %1 переводим выбранную задачу на передний план. После завершения задачи в переднем плане она исчезает из списка фоновых процессов, команда bg уже не сработает. 

Когда процесс выполняется в переднем плане, можно приостановить его нажатием Ctrl + Z. После этого процесс приостанавливается. Командой bg %1 запускаем приостановленную задачу в фоне, терминал становится свободным для других команд.
